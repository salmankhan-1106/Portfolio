# Muhammad Salman Khan — Portfolio

Personal portfolio site built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS** — styling
- **Framer Motion** — scroll and entrance animations
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
